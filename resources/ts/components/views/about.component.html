About

What's a Twist? It's a portmanteau; combining Twitter and List. Twists makes it easier to consume popular lists on Twitter without having to sign up or sign in.

Topics

Topics emerged from the collation of popular lists that were collected. If you feel that there should be a topic of interest, feel free to send a tweet or an email .

Lists

Lists are curated by popularity. Metrics used are previously listed popular lists by individuals and companies, number of members on a list, and number of subscribers to a list. If you feel that there should be a list in a certain topic, feel free to send a tweet or an email .

Colophon

This web application was written using the Angular front-end framework with a PHP Laravel back-end hosted by Heroku and a Redis data store hosted by Redis Labs.
