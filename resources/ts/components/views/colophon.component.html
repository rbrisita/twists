This is an example web application written using Angular with a PHP backend and Redis data store. It exposes Twitter List Timeines to unregistered users. Users can save their favorite topics locally with cookies. Due to Twitter's API polling, the list timelines are static for a better user experience. Refresh web page to get updated timelines.
